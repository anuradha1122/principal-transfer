# Module Delivery Structure

Each development module must contain:

1. Project folder command
2. Package installation
3. File creation commands
4. Migration
5. Model
6. Relationships
7. Form Requests
8. Controllers
9. Policies and authorization
10. Notifications
11. PDF and Excel exports
12. Seeders
13. Routes
14. Sidebar update
15. React pages
16. Documentation updates
17. Testing checklist
18. Automated tests
19. Git commit
20. Next exact step

All admin pages must use AdminLayout.

Public registration must remain disabled until NIC-controlled
registration is implemented.
