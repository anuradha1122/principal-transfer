# System Scope

## Included in MVP

- Authentication
- NIC-controlled principal registration
- Roles and permissions
- Seven-zone administrative structure
- Divisions and schools
- Principal profiles
- Transfer applications
- Zonal review
- Provincial review
- Transfer Board decisions
- Application status tracking
- PDF documents
- Basic email notifications
- Reports and Excel exports
- Audit trail

## Excluded from MVP

- Automated vacancy matching
- GIS distance calculations
- Transfer appeals
- SMS gateway
- Mobile application
- Payroll integration
- Advanced scoring algorithms
- Digital signatures
- Multi-province support
