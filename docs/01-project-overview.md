# Project Overview

## Project Name

Principal Transfer System

## Organization

Provincial Department of Education  
Sabaragamuwa Province, Sri Lanka

## Purpose

The system manages principal transfer applications from submission
through zonal review, provincial review and the final Transfer Board
decision.

## Technology Stack

- Backend: Laravel
- Frontend: React with Inertia.js
- Database: MySQL
- Authentication: Laravel Breeze
- Authorization: Spatie Laravel Permission
- Styling: Tailwind CSS
- Icons: Lucide React
- PDF: DomPDF
- Excel: Laravel Excel

## Main Users

- Super Admin
- Principal
- Zonal Director
- Provincial Director
- Transfer Board Member
- Data Entry Officer

## MVP Workflow

1. Principal registers after NIC verification.
2. Principal completes a profile.
3. Principal submits a transfer application.
4. Zonal Director approves or rejects.
5. Provincial Director approves or rejects.
6. Transfer Board records the final decision.
7. Principal views the final result.
