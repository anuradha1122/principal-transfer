# User Roles

## Super Admin

Has access to all system modules and settings.

## Principal

Can maintain an own profile, submit transfer applications and track
the final decision.

## Zonal Director

Can review applications originating from the director's assigned zone.

## Provincial Director

Can review applications approved at zonal level and send approved
applications to the Transfer Board.

## Transfer Board Member

Can view board-pending applications and record final decisions.

## Data Entry Officer

Can maintain organization, school and principal registry information.
